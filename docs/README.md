## What is AWS Calculator MCP?

A tool that converts plain English AWS infrastructure descriptions into official AWS Pricing Calculator links with real costs already computed.

```bash
aws-calc --prompt "2 t3.large EC2, RDS MySQL 100GB, 500GB S3"
→ $352.13/mo
→ https://calculator.aws/#/estimate?id=...
```

Works as:
- **CLI:** `aws-calc --prompt "your infra"`
- **Interactive:** `aws-calc -i` (asks questions)
- **MCP server:** for Claude, Cursor, VS Code
- **REST API:** self-hosted automation

## Features

✅ **~50 AWS services** — EC2, Lambda, RDS, DynamoDB, S3, Bedrock, OpenSearch, API Gateway, Cognito, KMS, CloudWatch, CloudTrail, WAF, VPC, and more

✅ **All major regions** — us-east-1, eu-west-1, ap-south-1, ap-southeast-1, us-west-2, etc.

✅ **Savings plans & commitments** — 1-year and 3-year commitment pricing for compute

✅ **Multi-service** — Build complex stacks in one prompt; auto-groups by category

✅ **Typo-tolerant** — Fuzzy matching: `lamda` → lambda, `buckit` → s3

✅ **Real AWS costs** — Uses the live calculator.aws, not estimates or approximations

✅ **Shareable links** — Generate permanent calculator.aws estimates others can open and verify

✅ **Multiple interfaces** — CLI, interactive mode, MCP server, REST API

✅ **Container-ready** — Works in Docker, GitHub Actions, serverless platforms
